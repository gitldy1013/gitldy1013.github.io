# 使用DockerCompose制作蘑菇博客YAML镜像文件

# 前言

首先特别感谢群里的小伙伴 @touch fish (opens new window) 使用DockerCompose部署了蘑菇博客，并且提供了 搭建文档 (opens new window)，本博客也是在上面的文档基础上进行修改~

本文主要讲解的是，如果将蘑菇博客制作成多个Docker业务镜像，逐步讲解每个镜像制作的过程

如果你只想快速部署蘑菇博客，那么可直接参考：DockerCompose一键部署蘑菇博客(Nacos版) (opens new window)

如果你想了解一下Docker Compose的使用，参考： Docker Compose入门学习 (opens new window)

如果你想把制作好的镜像提交到镜像仓库，参考：使用GithubAction构建蘑菇博客镜像提交DockerHub (opens new window)

如果你想了解Docker图形化工具Portainer的使用，参考：Docker图形化工具Portainer介绍与安装 (opens new window)

# 安装基础环境

下面我们到蘑菇博客的源码目录，运行 bash.sh 脚本，主要用于安装常用工具，node等环境，内容如下。

#!/usr/bin/env bash

echo "=====Linux必要软件安装====="

echo "=====安装vim net-tools unzip wget git maven====="
yum install vim net-tools unzip wget git maven -y

echo "=====安装node14====="
wget https://mirrors.tuna.tsinghua.edu.cn/nodejs-release/v14.15.1/node-v14.15.1-linux-x64.tar.gz

tar xf node-v14.15.1-linux-x64.tar.gz -C /usr/local

mv /usr/local/node-v14.15.1-linux-x64/ /usr/local/node14

echo '
export NODE_HOME=/usr/local/node14
export PATH=$NODE_HOME/bin:$PATH
' >> /etc/bashrc

source /etc/bashrc

ln -s /usr/local/node14/bin/npm /usr/local/bin/
ln -s /usr/local/node14/bin/node /usr/local/bin/

echo "=====node14版本====="
node -v

echo "=====npm 版本====="
npm -v

rm -rf node-v14.15.1-linux-x64

maven的依赖，默认是从 maven repository中央仓库拉取的，可能会比较慢，我们需要增加镜像地址+

<mirrors>
	<mirror>
        <id>alimaven</id>
        <mirrorOf>central</mirrorOf>

        <url>http://maven.aliyun.com/nexus/content/repositories/central/</url>
    </mirror>
    <mirror>
        <id>jboss-public-repository-group</id>

该命令主要执行了 mysql文件夹中的 docker-compose.yml ，然后给我们创建一个mysql容器

version: '3.1'
services:
  mysql:
    image: mysql
    restart: always
    container_name: mysql
    environment:
      MYSQL_ROOT_PASSWORD: root
    command:
      --default-authentication-plugin=mysql_native_password
      --character-set-server=utf8mb4
      --collation-server=utf8mb4_general_ci
      --explicit_defaults_for_timestamp=true
      --lower_case_table_names=1
    ports:
      - 3306:3306
    volumes:
      - ./data:/var/lib/mysql
      - ./init/:/docker-entrypoint-initdb.d/
    networks:
      - mogu
networks:
  mogu:
    external: true

# 增加这样一条配置，一个进程在VMAs(虚拟内存区域)创建内存映射最大数量

# 让配置生效
sysctl -p

# 重启 docker，让内核参数对docker服务生效
systemctl restart docker
